# ComentaPro 
Es una app para hacer publicaciones y comentar en ellas. Básicamente es como una red social pero simple.

¿Qué Hace?


- Crear publicaciones sobre cualquier tema
- Comentar en las publicaciones de otros
- Editar tus propias publicaciones y comentarios
- Borrar lo que escribiste

- Servicios
Authentication Service
Es un servicio que maneja login y usuarios. Está hecho en .NET.
- Puerto: 5001

Publication Service
Es donde está todo lo de publicaciones y comentarios. Está hecho en Node.js
- Puerto: 5001
- Base de datos: MongoDB
